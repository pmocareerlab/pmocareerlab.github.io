### H2：そもそもPMOの転職活動、エージェントなしでいけるか？

#### H3：求人サイト直接応募との違い
- 非公開求人へのアクセス
- 職務経歴書の添削サポート
- 企業との交渉代行（年収・条件）
- 面接フィードバックがもらえる

#### H3：PMOは「スキルの言語化」が特に難しい
- 「何をしたか」は言えても「何ができるか」が伝わりにくい
- エージェントの添削で「強みの言語化」ができる
- →ここに実体験を入れる（A4：「希望転職に対して強みになる書き方を指導してもらった」）

{{< note type="experience" title="職務経歴書の添削で気づいたこと" >}}
エージェントに職務経歴書を見てもらうまで、私は「担当業務の列挙」しかしていませんでした。「WBS管理・課題管理・ベンダー折衝」と書いても、それがどのくらいの規模で、何を解決したのかが伝わらない。「あなたが変えたことは何ですか？」という問いで初めて、自分のPMO経験の"価値ある部分"が見えてきました。
{{< /note >}}

### H2：PMO転職の現実——求人数と求人の種類

**← この記事の最大の差別化ポイント。他サイトは書いていない。**

#### H3：PMO専用求人は思ったより少ない

{{< note type="experience" title="実際に転職活動して気づいた求人の実態" >}}
登録後に気づいたのが、「PMO」と明記された求人は案外少ないということ。多くはPM・TL・プロジェクトリーダー募集。PMOとPMは似ているようで、企業側の期待値が違います。PMOは「管理支援」、PMは「責任者」。この違いを理解した上で求人を見ないと、見当違いの応募を続けることになります。
{{< /note >}}

#### H3：視野を広げないと選択肢が狭くなる
- PMO純正にこだわると件数が激減する
- PM・TL・ITコンサル・PdMまで範囲を広げると一気に選択肢が広がる
- エージェントに「PMO経験を活かせるポジションを広く」と伝えるのが正解

{{< voice speaker="reader" >}}
PMOとして転職したいのに、PM求人を紹介されるのは違う気がするんですが…
{{< /voice >}}

{{< voice speaker="owner" >}}
私も最初はそう思いました。でも実際に求人票を読むと、「プロジェクト管理全般・ステークホルダー調整・課題管理」という職務内容がPMOと重なるPM求人が多い。タイトルに縛られず、**中身を見て判断する**ようにエージェントに相談したら、選択肢が広がりました。
{{< /voice >}}

### H2：実際に使ったエージェント2社の正直な評価

**← 実体験ベースのメインコンテンツ**

---

#### H3：①リクルートエージェント——求人数は圧倒的。ただし絞り込みが必要

**良かった点：**
- 求人数が多く、選択肢の全体像が見える
- Web完結で登録しやすい
- 面談の雰囲気はリラックスして話せた

**気になった点：**
- 配信メールの量が多い（設定で調整可能）
- PMO専用求人よりPM寄りの求人が多い印象
- 求人の量に対して「これが自分に合う」と感じる件数は絞られる

{{< note type="experience" title="リクルートエージェントに登録してわかったこと" >}}
登録後すぐに大量の求人メールが届きます。最初は圧倒されましたが、条件を絞ったり担当者に「PMO経験を活かせるポジションで年収アップを希望」と明確に伝えることで、見るべき求人が整理されました。量の多さは武器ですが、使い方次第です。
{{< /note >}}

**こんな人向き：**
- まず市場の全体像を把握したい人
- 選択肢を広く持って比較したい人
- IT・PMO・PM系で幅広く見たい人

---

#### H3：②JACリクルートメント——高年収・厳選紹介。PMO経験者との相性は良い

**良かった点：**
- 今の年収からアップする案件を中心に紹介してくれた
- 厳選されている分、1件1件の求人が的外れになりにくい
- 担当者がしっかりヒアリングしてくれる

**気になった点：**
- 紹介件数はリクルートエージェントより少ない
- 求人の「幅の広さ」より「的中率の高さ」重視の設計

{{< note type="experience" title="JACリクルートメントを選んだ理由" >}}
リベラルアーツ大学（両学長）が紹介していたことで知りました。「年収アップを前提にした転職支援」という評判通り、紹介案件は基本的に今より収入が増える前提のものでした。件数は少ないですが、「これは的外れだな」と感じる求人もほぼなかった。求人の量より質を重視する人に向いていると思います。
{{< /note >}}

**こんな人向き：**
- 年収アップを明確な目標にしている人
- ミドル〜ハイクラス（年収600万円以上）の人
- 的外れな求人に時間を使いたくない人

---

#### H3：③ビズリーチ——転職意欲がなくても使える「市場価値の物差し」

**良かった点：**
- 受け取ったスカウトの年収帯で市場価値が可視化される
- 転職しなくても情報収集として使える
- 企業側から声がかかるため、思わぬ選択肢が見つかることがある

**使い方のポイント：**
- PMO・PM・ITプロジェクト管理系のキーワードをプロフィールに入れる
- スカウトの年収帯が「今の年収と比べてどうか」を確認するだけでも十分価値がある
- 「転職活動を開始する」覚悟がなくても登録してOK

---

### H2：2社を実際に使って感じた「複数登録のリアル」

#### H3：比較できることの価値

{{< note type="experience" title="2社並行して気づいたこと" >}}
リクルートエージェントとJACを同時に使うことで、同じ「PMO経験5年」でも、エージェントによって「刺さる求人のゾーン」が違うとわかりました。リクルートエージェントは量で網羅し、JACは年収軸で絞る——この使い分けは最初からわかっていたわけではなく、並行して使ってみて初めてわかったことです。1社だけだと比較の基準が持てません。
{{< /note >}}

#### H3：面談の流れ（両社共通）

1. **Web登録**（5〜10分）
2. **キャリアアドバイザーとの面談設定**（電話/オンライン）
3. **面談**（60〜90分）：重視することのヒアリング、転職軸の確認
4. **職務経歴書の添削・アドバイス**
5. **求人紹介スタート**

{{< note type="info" title="面談は思ったよりリラックスして受けられます" >}}
エージェントの面談というと身構えますが、実際はキャリア相談に近い雰囲気です。「転職を絶対にしなければ」という空気にはなりませんでした。現状の不満・希望年収・働き方の優先度などを自由に話せます。「転職を迷っている段階」で来る人は珍しくないと担当者も言っていました。
{{< /note >}}

---

### H2：転職活動を経て、私が気づいたこと

**← ここが記事の最大の差別化。正直な体験談。**

{{< note type="experience" title="転職活動で「本当の問題」が見えてきた" >}}
転職エージェントに登録して、面談を重ね、求人を見続けた末に気づいたのは——「自分が感じていた閉塞感は、職場の問題だけじゃない」ということでした。

市場を知ることで「今の自分に何ができるか」「外から見てどう映るか」が客観的にわかった。その上で冷静に考えたとき、今の環境を変えることよりも先にやるべきことがある、という結論に至りました。今回は転職を見送りましたが、それは「転職しなかった」のではなく「転職が必要かどうかを、ちゃんと判断できた」ということだと思っています。

転職エージェントへの登録は、「転職する人のためのもの」ではなく、「自分のキャリアを客観視したい人のためのもの」でもあります。
{{< /note >}}

{{< voice speaker="reader" >}}
転職しないとわかったなら、エージェントに登録する意味はなかったのでは？
{{< /voice >}}

{{< voice speaker="owner" >}}
逆です。登録したことで「今は転職しない」という**能動的な判断**ができました。何も動かなければ、閉塞感を抱えたまま「いつか動こう」と先送りし続けていたと思います。市場を知った上での「残留」は、何も知らずにいる「惰性での残留」とは全然違います。
{{< /voice >}}

### H2：PMO転職でエージェントを選ぶ3つの軸

1. **求人数の幅**：まず選択肢を広く持ちたいか（→リクルートエージェント）
2. **年収軸**：今より確実にアップしたいか（→JAC）
3. **市場価値の可視化**：まず自分の相場を知りたいか（→ビズリーチ）

### H2：よくある質問

**Q：登録したらすぐ転職活動が始まりますか？**
→ No。登録後は面談があり、そこで「今すぐ動く」「情報収集段階」を伝えられます。担当者に正直に話せばペースを合わせてくれます。

**Q：在職中でも登録できますか？**
→ Yes。むしろ在職中の登録が基本です。エージェントは在職中転職のサポートを前提に動いています。

**Q：PMO経験が伝わらないことはありますか？**
→ あります。PMOという職種を正確に理解していない担当者もいます。「PMO＝プロジェクト管理支援、ステークホルダー調整、課題管理の専門職」と最初に説明するのが有効です。

**Q：複数登録は迷惑ではないですか？**
→ 問題ありません。業界では当たり前の行動で、エージェント側も想定済みです。

---

### H2：まとめ——エージェントに登録するのは「転職する決意をしてから」ではない

結論：**「転職しようか迷っている段階」こそ登録のタイミングです。**

市場を知ることで、転職するかしないかの判断が初めて客観的にできます。登録は3〜5分、相談は無料、転職しなくても問題なし。この条件でやらない理由はありません。
